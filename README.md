## 機能
- ✅ Excel (.xlsx) およびWord (.docx) ファイルの翻訳
- ✅ 12言語以上に対応
- ✅ カスタム翻訳指示のサポート
- ✅ レイアウトと書式の保持

  ## 必要な設定
※無料APIのレート制限がある為,1日と時間あたりのリクエスト数を確認してください。
### 1. Gemini APIキーの取得
1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセス
2. Googleアカウントでログイン
3. 「Create API Key」をクリック
4. 生成されたAPIキーをコピー

### 2. プラグイン設定

1. Exmentのプラグイン設定画面を開く
2. 「AI翻訳システム」の設定を開く
3. 「Gemini APIキー」欄にコピーしたAPIキーを貼り付け
4. 設定保存
